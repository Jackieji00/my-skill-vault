# Morning Routine (每日晨间工作流)

该技能用于自动执行每日开启工作时的重复性任务，确保信息同步、视频处理和标签归档。

## 核心流程

### 1. 论文推荐 (Bilingual Enhancement)
- **指令**：首先运行 `start-my-day` 技能。
- **增强要求**：
    - 检查生成的今日论文笔记（通常在 `10_Daily/` 目录下）。
    - 确保笔记内容为**中英双语对照**版本（包含标题、摘要、核心贡献等）。
    - 若原始输出仅为单语，请调用 AI 处理能力将其转换为双语格式（参考 `10_Daily/2026-06-05-论文推荐-双语版.md` 的排版风格）。

### 2. 待办视频自动处理 (#pending/video)
- **扫描**：在 Obsidian 库中搜索包含标签 `#pending/video` 的所有笔记。
- **循环处理每一篇笔记**：
    1. **提取信息**：从 Frontmatter 的 `url` 字段提取视频链接。
    2. **加载规范**：读取根目录（或 `_templates/`）下的 `_tags_index.md`。理解其中的标签体系、层级（如：求职、算法、技术栈）及归并规则（如：避免空格、优先用英文算法名）。
    3. **自动转录**：调用 `bilibili-auto-transcript` 技能处理提取的 URL。
    4. **智能标签匹配**：
        - 根据视频转录内容，**优先匹配** `_tags_index.md` 中已有的核心标签。
        - 若内容涉及新领域，则参照规范创造新标签（小写、连字符、无空格）。
        - 必须包含维度：主题、技术点、行业（如果是求职相关）。
    5. **回写笔记**：
        - 将生成的转录摘要和结构化内容写入原笔记。
        - 更新 Frontmatter 中的 `tags` 数组。
        - 将 `#pending/video` 修改为 `#completed/video`（或按照库内习惯标记为已完成）。

### 3. 首页同步与归档管理 (HomePage & Archive Management)
- **“飞行甲板”定义**：`HomePage.md` 是当天的核心中控台，仅保留“今天”正在处理的信息和汇总，不堆积历史。
- **归档逻辑 (仅限 HomePage 和 10_Daily 论文笔记)**：
    1. **检查日期**：读取 `HomePage.md` 顶部的日期。
    2. **旧内容迁移**：
        - 若日期**不是今天**：
            - 计算该日期所属的年、月、周（例如：2026/06/Week-01）。
            - 将旧的 `HomePage.md` 内容存入 `10_Daily/年/月/周/HomePage-Archive-日期.md`。
            - 将 `10_Daily/` 下非本周的旧论文笔记移动到对应的 `10_Daily/年/月/周/` 目录下。
            - 清空 `HomePage.md` 并写入今日日期。
        - 若日期**是今天**：执行**增量合并**。

### 4. 视频笔记专项处理 (VideoNotes Routing)
- **存储路径**：所有经由 `bilibili-auto-transcript` 处理生成的视频笔记，必须统一存放在 `VideoNotes/` 文件夹下。
- **关联更新**：处理完成后，在 `HomePage.md` 的“今日进展”中列出这些新生成的笔记链接。

### 5. 全局变动汇总 (Workspace Change Summary)
- **动态追踪**：记录执行 `morning-routine` 期间发生的所有文件变动（包括 `VideoNotes/` 的更新、`10_Daily/` 的生成、`_tags_index.md` 的修改）。
- **汇总写入**：在 `HomePage.md` 中以列表形式展示：
    - 📚 **今日论文**：[[10_Daily/日期-论文推荐|论文标题]]
    - 🎬 **视频笔记**：[[VideoNotes/笔记名|视频标题]]
    - 🏷️ **标签/规范更新**：索引文件变动及新增标签列表。
    - 📂 **归档清理**：哪些文件被移动到了哪个周文件夹。

## 注意事项

- **目录隔离**：`VideoNotes/` 是静态存储区，不参与 `10_Daily/` 的年/月/周归档逻辑。
- **标签严谨性**：优先复用索引标签。更新 `_tags_index.md` 时需保持其 YAML 结构完整。
- **目录自动创建**：执行归档或移动时，若目标目录不存在，请自动递归创建。


## 示例 (HomePage.md 更新格式)

```markdown
## 📅 2026-06-05 工作汇总

### 📚 论文推荐
- [[10_Daily/2026-06-05-论文推荐-双语版|今日论文推荐 (中英双语)]]

### 🎬 视频处理 (#completed/video)
- [x] [[笔记名|视频标题]] - 已完成转录与标签匹配
- [x] [[笔记名|视频标题]] - 已完成转录与标签匹配

### 🏷️ 标签变动
- 新增标签：`#new-technology`, `#industry-insight`
```
